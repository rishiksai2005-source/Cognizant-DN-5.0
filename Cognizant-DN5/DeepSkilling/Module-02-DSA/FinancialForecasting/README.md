# Exercise 7: Financial Forecasting

## Objective

Predict future financial value using Recursion.

## Language

Java

## Formula

Future Value = Current Value × (1 + Growth Rate)^Years

## Files

- FinancialForecast.java
- Main.java

## Output

```
Current Value : 10000.0
Growth Rate   : 10.0%
Years         : 5
```
