# Lab 05 - Retrieve Data using Entity Framework Core

## Objective
Retrieve product records from SQL Server using Entity Framework Core.

## Technologies Used
- C#
- .NET 10
- Entity Framework Core
- SQL Server

## Features Implemented
- Retrieve all products
- Retrieve product by ID
- Retrieve products using LINQ
- Display product details in the console

## Result
Successfully connected to SQL Server and retrieved product records using Entity Framework Core.

## Output
- Displayed Product ID
- Displayed Name
- Displayed Price
- Displayed Stock
- Displayed Category
