# Handson 2 - CRUD Web API

## Objective

Implement CRUD operations using ASP.NET Core Web API.

## Technologies Used

- ASP.NET Core Web API
- .NET 10
- Swagger (OpenAPI)

## Features

- GET All Products
- GET Product by ID
- POST New Product
- PUT Update Product
- DELETE Product

## Concepts Covered

- API Controllers
- Routing
- HTTP Methods
- IActionResult
- Status Codes
- Swagger Testing

## Result

Successfully created and tested a CRUD Web API using an in-memory product list.
